# Wictor Films

Site institucional premium para captação de clientes da produtora audiovisual Wictor Films.

## Site publicado

https://wictor-films.netlify.app

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Segurança

Arquivos locais e sensíveis ficam fora do GitHub via `.gitignore`, incluindo `.env`, `.env.*`, `node_modules`, `dist`, `.netlify` e logs de desenvolvimento.
